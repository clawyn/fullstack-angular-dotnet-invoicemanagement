InvoiceManagement:
InvoiceManagement is a web application to manage products, customers, and invoices. I coded this project to practice and learn during my internship.

Features:
Manage products, customers, invoices: create, read, update, delete (CRUD)
Automatic calculation of the total invoice amount
Option to add extra fees
View all invoices for each customer
User authentication and security with JWT
Password hashing with Argon2
API documentation with OpenAPI

Technologies:
Back-end: C# 12, .NET 8, ASP.NET Core, EF Core (PostgreSQL)
Security: JWT, Argon2
API Documentation: OpenAPI
Front-end: Angular 19, TypeScript, Angular Material, PrimeNG, RxJS, Angular Router
